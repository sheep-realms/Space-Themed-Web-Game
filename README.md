# 某个太空主题的网页游戏（未命名）
这是一个以太空航行为主题的网页小游戏，游戏目前的开发进度为：刚刚起步。

## 文件结构
```
doc/ - 开发文档
lang/ - 本地化
lib/ - 支持库
res/ - 资源
├ audio/ - 音频
│ ├ music/ - 音乐
│ └ se/ - 音效
├ class/ - 类
├ cursor/ - 鼠标指针
├ data/ - 数据
│ ├ arms/ - 武器
│ ├ base/ - 基建设施
│ ├ map/ - 地图
│ └ sound.json - 音频数据
├ image/ - 图像
│ └ icon/ - 图标
├ script/ - 脚本
└ style/ - 样式表
index.html - 首页
```
